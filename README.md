###  프로그램 이름 
> 😇 사실 아직 정한게 없다.. 뭔가 삘 꽂히는게 없음ㅠ (고민중)

- **Life Disc**
    - 생활속에서 허리 디스크(Disc)를 항상 생각하자는 뜻에서
- **Disk Disc** (DD / 디디)
    - 하드 디스크의 Disk와 허리 디스크의 Disc를 이용한 말장난
        
        줄여서 DD로 부를수도 있다!
        

- ~~**Human Disk Disc** (HDD)~~
    - 컴퓨터의 하드디스크를 뜻하는 HDD의 약자를 이용한 말장난
        
        오히려 HDD의 용어에 혼란이 올 거 같아 취소
        
- 꼬북이
    - 거북목의 거북이 단어를 이용
